# Commands
- web 앱 dev 서버
    ```shell
    npm run dev:web
    ```

- web 앱 build
    ```shell
    npm run build:web
    ```

- 전체 lint
    ```shell
    npm run lint
    ```

- 전체 format
    ```shell
    npm run format
    ```

- 특정 workspace에 패키지 설치
    ```shell
    npm install <package name> --workspace web
    ```
